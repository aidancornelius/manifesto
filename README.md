# Manifesto

A personal manifesto site at [manifesto.ursa.nz](https://manifesto.ursa.nz). Each page is a numbered principle with its own colour scheme, navigable in sequence. Built with Astro and deployed to GitHub Pages.

Manifesto entries live as markdown files in `src/content/manifesto/`. To add a new one, create a markdown file with frontmatter for `number`, `title`, and light/dark colour values.

Run `npm install` then `npm run dev` for local development.
